# アークナイツ ダメージ計算式まとめ

## 目的

このファイルは、アークナイツダメージ計算アプリで最終的に使う計算式をまとめたもの。

まず ver1.0 では「通常攻撃1回のダメージ」を実装し、あとからスキル倍率、攻撃速度、DPS、総ダメージを追加する。

## 1. 基本用語

| 用語 | 意味 |
|---|---|
| 攻撃力 | オペレーターの最終攻撃力 |
| 敵防御力 | 物理ダメージを固定値で減らす値 |
| 敵術耐性 | 術ダメージを割合で減らす値。例: 術耐性20なら術ダメージは80%になる |
| 最低保証ダメージ | 物理・術ダメージでは、攻撃力の5%を最低値とする |

## 2. 通常攻撃1回の基本式

### 2.1 物理ダメージ

```text
物理ダメージ = max(攻撃力 - 敵防御力, 攻撃力 * 0.05)
```

例:

```text
攻撃力610、敵防御力300の場合

610 - 300 = 310
攻撃力の5% = 30.5

物理ダメージ = 310
```

### 2.2 術ダメージ

```text
術ダメージ = 攻撃力 * max(100 - 敵術耐性, 5) / 100
```

例:

```text
攻撃力612、敵術耐性20の場合

100 - 20 = 80
612 * 80 / 100 = 489.6

術ダメージ = 489.6
```

アプリで整数表示する場合:

```python
術ダメージ = int(612 * 80 / 100)
# 489
```

### 2.3 確定ダメージ

```text
確定ダメージ = 攻撃力
```

確定ダメージは、基本的に敵防御力と敵術耐性の影響を受けない。

ただし、最終ダメージ軽減、バリア、シールドなどを扱う場合は別途補正する。

## 3. 補正込みの最終式

将来的にスキル、素質、バフ、デバフを入れる場合は、以下の順番で考える。

### 3.1 最終攻撃力

```text
最終攻撃力 =
  floor(基礎攻撃力 * ステージ補正) * (1 + 攻撃力割合バフ合計)
  + 攻撃力固定値バフ合計
```

攻撃力低下デバフがある場合:

```text
最終攻撃力 =
  floor(上の攻撃力 * 攻撃力低下補正の積)
```

### 3.2 スキル倍率込みの攻撃力

「攻撃力+50%」のような効果:

```text
攻撃力 = 基礎攻撃力 * 1.5
```

「攻撃力の200%のダメージ」のような効果:

```text
ダメージ計算用攻撃力 = 最終攻撃力 * 2.0
```

注意:

| 表現 | 扱い |
|---|---|
| 攻撃力+X% | 攻撃力そのものを増やす |
| X%のダメージ | ダメージ計算時の倍率として扱う |

### 3.3 補正後の敵防御力

```text
補正後防御力 =
  max(0, (敵防御力 + 防御固定値補正) * (1 + 防御割合増加) * 防御割合低下の積)
```

例:

```text
敵防御力1000、防御-30%の場合

1000 * (1 - 0.30) = 700
```

### 3.4 補正後の敵術耐性

```text
補正後術耐性 =
  clamp((敵術耐性 + 術耐性固定値補正) * (1 + 術耐性割合増加) * 術耐性割合低下の積, 0, 100)
```

例:

```text
敵術耐性50、術耐性-20の場合

50 - 20 = 30
```

### 3.5 最終物理ダメージ

```text
物理ダメージ =
  max(ダメージ計算用攻撃力 - 補正後防御力, ダメージ計算用攻撃力 * 0.05)
```

最終ダメージ増加・軽減を入れる場合:

```text
最終物理ダメージ =
  物理ダメージ * ダメージ増加倍率 * ダメージ軽減倍率
```

### 3.6 最終術ダメージ

```text
術ダメージ =
  ダメージ計算用攻撃力 * max(100 - 補正後術耐性, 5) / 100
```

最終ダメージ増加・軽減を入れる場合:

```text
最終術ダメージ =
  術ダメージ * ダメージ増加倍率 * ダメージ軽減倍率
```

### 3.7 最終確定ダメージ

```text
確定ダメージ =
  ダメージ計算用攻撃力
```

最終ダメージ増加・軽減を入れる場合:

```text
最終確定ダメージ =
  確定ダメージ * ダメージ増加倍率 * ダメージ軽減倍率
```

## 4. 攻撃速度とDPS

### 4.1 攻撃間隔

```text
最終攻撃間隔 =
  基礎攻撃間隔 * 100 / (100 + 攻撃速度補正)
```

攻撃間隔を直接変える効果がある場合:

```text
最終攻撃間隔 =
  (基礎攻撃間隔 + 攻撃間隔固定補正) * 100 / (100 + 攻撃速度補正)
```

### 4.2 1秒あたりの攻撃回数

```text
1秒あたりの攻撃回数 = 1 / 最終攻撃間隔
```

### 4.3 DPS

```text
DPS = 1回あたりの最終ダメージ / 最終攻撃間隔
```

複数回ヒットする攻撃の場合:

```text
DPS = 1回あたりの最終ダメージ * ヒット数 / 最終攻撃間隔
```

## 5. スキル中の総ダメージ

### 5.1 持続時間型スキル

```text
攻撃回数 = floor(スキル持続時間 / 最終攻撃間隔)
総ダメージ = 1回あたりの最終ダメージ * 攻撃回数
```

### 5.2 弾薬型スキル

```text
攻撃回数 = 弾薬数
総ダメージ = 1回あたりの最終ダメージ * 弾薬数
```

### 5.3 単発型スキル

```text
総ダメージ = 1回あたりの最終ダメージ * ヒット数
```

## 6. ver1.0で実装する範囲

ver1.0では、まず以下だけを実装する。

- 通常攻撃1回の物理ダメージ
- 通常攻撃1回の術ダメージ
- 敵防御力
- 敵術耐性
- 最低保証ダメージ5%

まだ実装しないもの:

- スキル倍率
- 攻撃速度
- DPS
- 総ダメージ
- 防御デバフ
- 術耐性デバフ
- 確定ダメージ
- 複数ヒット
- 職分特性
- 素質
- モジュール

## 7. 現在のPython実装に対応する式

現在の `damage_calculator.py` では、以下の式を使う。

物理:

```python
minimum_damage = int(atk * 0.05)
damage = max(int(atk - enemy_def), minimum_damage)
```

術:

```python
damage_rate = max(100 - enemy_res, 5) / 100
damage = int(atk * damage_rate)
```

通常攻撃:

```python
if damage_type == "physical":
    damage = calculate_physical_damage(atk, enemy_def)

if damage_type == "arts":
    damage = calculate_arts_damage(atk, enemy_res)
```
